## Overview
This project documents the investigation of a website compromise caused by a brute force attack against an administrative account.

The attacker embedded malicious JavaScript into the website, prompting users to download a file that redirected them to a malicious domain.

## Objectives
- Analyze network traffic using tcpdump
- Identify network protocols involved in the attack
- Determine root cause
- Recommend security controls to prevent brute force attacks

## Skills Demonstrated
- Network protocol analysis (DNS, TCP, HTTP)
- Brute force attack identification
- Incident response documentation
- Root cause analysis
- Security control recommendations
