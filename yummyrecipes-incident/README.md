# Cybersecurity Incident – YummyRecipesForMe.com

## Overview
This project demonstrates the investigation of a network outage affecting `www.yummyrecipesforme.com`.  
Several users reported being unable to access the website, receiving the error **“destination port unreachable”**.  

This report includes:

- TCPDump log analysis  
- Protocols involved  
- Issue interpretation  
- Investigation steps  
- Root cause and recommended solution

## Objective
- Analyze network traffic to identify the affected protocol  
- Interpret ICMP error messages related to DNS queries  
- Propose remediation and preventive measures
